# 本章主要介绍python的大概
> 在越来越多人用python3的时代里，我们没有理由再去了解python2版本，因为他们并不兼容

*  python2 和python3 很不同，在Mac上默认输入python显示的是版本2，所以当我们用3的时候我们需要用

~~~
python3 --version //标出是3 否则使用
python --version //默认显示的版本2
~~~

* 我们使用的python实际上是由解释器来解释的，就像js解释器一样，有很多解释器，著名的有v8，所以说我们会看到很多解释器，不过我建议大家选用官方默认的cpython解释器，用户是最多的
* python默认的文件后缀名是.py
